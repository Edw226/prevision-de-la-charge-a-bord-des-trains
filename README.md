# prevision-de-la-charge-a-bord-des-trains
il a été mis à notre disposition :
-un fichier csv avec les données d'entrainement et de test;
-une rapide description des champs du fichier.

L'ojectif etait de prédire la charge à bord des trains sur une semaine,
en essayant de faire en sorte, autant que faire se peut, que la méthode choisie puisse 
être implémentée en temps réel.
Nous avons ainsi proposé plusieurs visualisations, ensuite nous
avons effectué une modélisation et enfin proposer des prévisions.
